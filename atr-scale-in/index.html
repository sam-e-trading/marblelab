Assumptions

  1. Gross scaling model: P&L per entry = max(total move ATR - entry ATR, 0) / stop distance ATR R.
  2. Equal unit size: every entry uses the same initial unit size. One unit stopped at the chosen stop distance equals -1R.
  3. Scale-in count: scale-ins exclude the initial entry at 0 ATR.
  4. Not a trade-management simulator: real stop movement, slippage, spread, partial exits, skipped fills, and position limits can change the result. A breakeven-stop simplification can be useful later, but this v1 keeps the path out of it.
