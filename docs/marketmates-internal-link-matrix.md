# MarketMates → Marble Lab Internal Link Matrix

Purpose: help Google and users understand that Marble Lab tools are MarketMates-owned practical tools connected to the relevant MarketMates lessons/articles.

Use this as an editorial implementation sheet for WordPress/internal linking. Prioritise links where the tool genuinely helps the reader take the next step.

## General rules

- Add 1–3 Marble Lab links per article/lesson where relevant. Avoid turning pages into link soup.
- Prefer contextual links inside body copy, not only footer/sidebar blocks.
- Use descriptive anchors like `MarketMates Position Sizing Calculator`, not vague `click here`.
- Where a lesson already mentions a calculator/spreadsheet/tool, replace or extend that mention with the Marble Lab link.
- Add reciprocal links from Marble Lab tool pages back to the strongest MarketMates education pages.
- Keep disclaimers: educational only, not financial advice.

# Notes / cautions

- Do not overuse exact-match anchors every time. Mix branded and natural anchors.
- Best anchor pattern: 50% branded (`MarketMates Position Sizing Calculator`), 30% descriptive (`position sizing calculator`), 20% natural sentence anchors (`calculate your 1R risk`).
- Avoid linking every tool from every article. Relevance beats volume.
- For SEO, the most important signal is from the already-ranking MarketMates education pages to the Marble Lab tool pages.
